# ComplyCo - Sample Applications

The purpose of this project is to showcase solutions and best practices on how to implement ComplyCo's private Recoder SDK.

This repository is also responsible for hosting the SDK packages in GitHub Packages, for the Maven repository to leverage as needed.

## Sample Apps

### `:sample-xml`

Showcase how to implement the **ComplyCo Recorder SDK** into a traditional Android application, using the old View-system with XML layouts.

This solution leverages the `com.complyco.recording-xml` package.

### `:sample-compose`

Showcase how to implement the **ComplyCo Recorder SDK** into an Android application using Jetpack Compose.

This solution leverages the `com.complyco.recording-compose` package.
