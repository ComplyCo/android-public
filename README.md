# ComplyCo - Sample Applications

The purpose of this project is to showcase solutions and best practices on how to implement ComplyCo's SDK (**ComplySDK**).

This repository is also responsible for hosting the SDK packages in GitHub Packages, for the Maven repository to leverage as needed.

## Sample Apps

### `:sample-xml`

Showcase how to implement the **ComplySDK** into a traditional Android application, using the old View-system with XML layouts.

This solution leverages the `com.complyco.complysdk.recording-xml` package.

### `:sample-compose`

Showcase how to implement the **ComplySDK** into an Android application using Jetpack Compose.

This solution leverages the `com.complyco.complysdk.recording-compose` package.
